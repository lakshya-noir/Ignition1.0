# rider_telemetry

## Requirements:
• Have flutter installed
• Install all the required libraries
• Make sure your mobile is connected to your laptop through a USB cable
• Make sure that the phone that you are using is in developer mode, and has Debugging via USB enabled
• Once you find your device in the list, then you can confirm that it has been connected to your list

## How to run:
• Use the command flutter run to run the code. 
• The app gets installed on the mobile device and is ready to be used.
